Blinkt Dashboard Help

This page contains information on how to use this Blinkt Dashboard utility.

Using the web dashboard

To use the web dashboard, you should head to the dashboard. You will then see buttons with preset colours. By clicking on them, the Blinkt will change to that colour.

Invoking through web requests

In a web browser

You can also invoke the Blinkt webapp through web requests.
You can head to http://[YOURDEVICEIP]/[HEXVALUE] in a web browser where you replace [YOURDEVICEIP] to the ip of the device running the webapp and [HEXVALUE] to a hexadecimal value of the colour you want to set the Blinkt to. You can find several hex colour pickers on the internet.
Please remeber to not include the # in the URL.

HTTP GET Requests

You can also change the Blinkt to a certain colour by sending a HTTP GET request to the same URL as above.
PLEASE NOTE: HTTP POST requests are not currently supported.
